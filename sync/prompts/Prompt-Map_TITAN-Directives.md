## 🔣 Prompt Categories

| Prompt        | Symbol | Function Type     | Description                                                              |
|---------------|--------|-------------------|--------------------------------------------------------------------------|
| `START`       | 🟢     | Initiator         | Begins the cycle; sets intent, initializes reflection or creation        |
| `UPDATE`      | 🔁     | Adaptive Feedback | Mid-cycle sync; tracks momentum, logs new insights or status             |
| `STOP`        | 🔴     | Closure/Reflection| Ends the cycle, logs conclusions, anchors narrative for archival          |
| `onTask`      | 🛠️     | Functional        | Actively engaged operational phase                                       |
| `onQuery`     | ❓     | Exploratory       | Invokes contextual insight or retrieval request                          |
| `onEOD`       | 🌙     | End of Day        | Wraps up day-level logs, syncs identity, numerology, and results         |
| `FinalAnswer` | 🔑     | Gnosis Trigger    | Marks moment of revelation or knowledge convergence                      |
| `SyncNow`     | 📡     | Protocol Layer    | Forces context realignment between agents, time, or identities           |
